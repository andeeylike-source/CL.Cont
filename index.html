⚙️ Настройки

👤 Пользовательские настройки

Переключай язык интерфейса и тему оформления. Выбор сохраняется автоматически.

🔑 API ключи

Ключи хранятся только в твоём браузере — не в коде и не на GitHub. Получить ключ: openrouter.ai/keys (бесплатно, модель gemini-2.0-flash-lite)

🔗 Алиасы ников

Если AI читает ник неправильно — укажи соответствие. Например: НигативВ → НигатиВ

⚖️ Веса ценности

Настрой формулу расчёта "ценности" для каждой роли. Сумма весов должна быть 100%.

🔔 Уведомления

Настройки уведомления при завершении анализа скриншотов.

💾 Экспорт / Импорт данных

Сохраняй данные в файл и загружай обратно. Удобно для резервного копирования и переноса.

🗑️ Данные

Сбросить все данные приложения (ростер, статистика, события). Ключи сохранятся.

Добро пожаловать в Clan Control

Инструмент офицера для отслеживания КМ-статистики клана. Давай настроим за 3 шага.

👥
1. Добавь игроков
Заполни ростер клана — ники, профы, паки
📅
2. Отметь КМ в Календаре
Открой день КМ и загрузи скриншоты для анализа
📊
3. Смотри статистику
История КМ, рейтинги, профили игроков
Также можно добавить API-ключ в Настройках для автоматического анализа скриншотов через AI
